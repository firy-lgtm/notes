
```sh
#1.网卡获取ip的方式2种
##1.1自动获取dhcp
##1.2手动设置(固定ip) manual手动(安装系统,nmtui中)

#B00TPROTO=dhcp 自动获取ip,

B00TPROTO=none
```

### 2.2 修改麒麟网卡配置文件并生效

```sh
#进入网卡配置文件，修改网卡
vim /etc/sysconfig/network-scripts/ifcfg-ens33

#修改ip后，通过下面命令重启网卡
ifdown ens33 && ifup ens33

#使用systemctl重启网卡
yum install -y network-scripts
#然后
systemctl restart network
```

## 六. /proc/目录

process进程信息.

1. /proc/存放的是各种服务软件进程信息,内核信息(配置)

2. /proc/目录相当于是给内存开了个入口.进程的内存信息.

3. 虚拟目录,不占用磁盘空间.仅仅入口.

4. /proc/展示的是内存中的信息,目录本身不占用磁盘空间.

5. 一切皆文件.

    

   

内存:运行内存,运行程序软件的地方.

磁盘/硬盘:永久存放数据.

| /proc/目录                             | 说明                           | 命令  |
| -------------------------------------- | ------------------------------ | ----- |
| /proc/数字命名的目录(只有数字)系统信息 | 进程信息,数字是进程的编号(PID) | ps    |
| /proc/cpuinfo  文件                    | cpu硬件信息                    | lscpu |
| /proc/meminfo 文件                     | 内存信息                       | free  |
| /proc/loadavg  文件                    | 负载信息(linux系统繁忙程度)    | w     |
