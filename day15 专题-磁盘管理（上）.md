# day15 专题-磁盘管理

## 一、磁盘管理

### 1.1磁盘

- 磁盘/硬盘：永久存放数据
- 内存：临时，程序软件运行地方



- 磁盘分类：

| 机械硬盘 | 价格低   hdd                   |
| -------- | ------------------------------ |
| 固态硬盘 | 性能高   ssd     8TB     128TB |

- 硬盘接口：

  | 磁盘接口   | 硬盘               | 大小                                                         |
  | ---------- | ------------------ | ------------------------------------------------------------ |
  | sata       | 机械硬盘，固态硬盘 | 机械：4tb   10k  性能要求不高地方，备份                      |
  | sas        | 机械硬盘，固态硬盘 | 机械：900G   15k   性能好   企业环境使用                     |
  | pcie-e，U2 | 固态硬盘           | tb级别   3tb    4tb   8tb   关键地方使用，性能，容量不高，数据库，存储服务 |

- 组成

  ```sh
  盘头   磁头（读写数据）
  磁道  扇区  柱面
  ```

- 固态硬盘颗粒

```sh
控制芯片：主控
存储颗粒：slc mlc tlc xxxx 3d nand
```

### 1.2磁盘分区流程

```sh
fdisk -l 查看分区，硬盘  /dev/ device 设备目录
fdisk /dev/xxxx 硬盘名字

p	输出磁盘信息，分区信息
q	退出不保存
w	退出并保存
n	创建分区
d	删除分区
```

### 1.3磁盘分区全流程

| 流程         | 说明                                               | 理解                                                   |
| ------------ | -------------------------------------------------- | ------------------------------------------------------ |
| 磁盘分区     | fdisk实现                                          | 划分隔断                                               |
| 磁盘格式化   | 也叫创建文件系统                                   | 给房间进行装修<br />不同的文件系统就是不不同的装修风格 |
| 磁盘分区挂载 | 分区设置入口，通过入口使用。<br />入口一般叫挂载点 | 安装门窗，人可以进入                                   |

```sh
#1.磁盘分区与检查
fdisk -l /dev/sdb

#2.格式化
mkfs.xfs /dev/sdb1

#3.挂载
mount /dev/sdb1  /mnt/

#4，测试
df -h /mnt/  # 属于 /dev/sdb1才对

touch /mnt/lidap.txt
```

永久挂载：

- 方案1：写入到/etc/fstab文件中。开机自动挂载

- 方案2：书写挂载命令，写入到/etc/rc.local

| /etc/rc.local文件每列 | 说明                          |
| --------------------- | ----------------------------- |
| 第一列                | 设备名字/分区名字/UUID        |
| 第二列                | 挂载点（目录）新建目录        |
| 第三列                | 文件系统类型xfs，ext4，swap   |
| 第四列                | 默认写defaults，挂载参数/选项 |
| 第五列                | 0 是否dump导出与备份          |
| 第六列                | 0 是否fsck进行磁盘检查        |

```sh
/dev/sdb1  永久挂载到/diy-data/ 目录.
/dev/sdb1    /diy-data/    xfs   defaults    0  0
```
