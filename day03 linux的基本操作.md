## 一. linux快捷键

| 常用快捷键 | 说明                         |
| ---------- | ---------------------------- |
| ctrl + c   | 取消当前操作                 |
| tab        | 自动补全命令                 |
| ctrl + a   | 光标移动到开头               |
| ctrl + e   | 光标移动到结尾               |
| ctrl + l   | 清屏                         |
| esc + .    | 使用上一个命令的最后一个内容 |

| 有坑的快捷键       | 说明                     |
| ------------------ | ------------------------ |
| ctrl + s           | 锁屏                     |
| ctrl + q或ctrl + c | 解锁                     |
| ctrl + z           | 后台挂起，执行fg命令恢复 |

## 二. 关机重启指令

| 命令     | 说明                                                         |
| :------- | ------------------------------------------------------------ |
| shutdown | shutdown -h 10 10分钟后关机<br />shutdown -c 取消<br />shutdown now 立马关机<br />shutdown -r 10 10分钟后重启 |
| poweroff | 关机，正常关机                                               |
| reboot   | 重启                                                         |

## 三. vi/vim的使用

### 3.1 基本用法

```sh
#编辑指定文件
vim /oldboy/oldboy.txt
#进入编辑（插入）模式
按英文i
#退出编辑（插入）模式
按esc键
#保存退出
按shift + ：键出现：符号
：wq 保存退出
：q！ 强制退出
#查看文件内容
cat /oldboy/oldboy.txt
```

### 3.2 常用的vim快捷键

```sh
# 先创建一个vim文件
cat /etc/services   /etc/sysconfig/network-scripts/ifcfg-ens33   > /oldboy/vim.txt
# 快速移动光标
文件第一行 gg
文件最后一行 G(shifg + g)
文件的某一行 100gg
#显示行号
set nu
#取消行号
set nonu
#搜索
/内容
向下 n
向上 N
# 可视行
V（可用于批量删除或复制多行）
#复制粘贴
y 复制当前行
p 粘贴
3p 粘贴3次
d 剪切当前行（删除）
C 把从光标处开始往后一行的内容删除并进入插入模式
#撤销
u
3u 撤销3次
```

## 四. 命令补充

### 4.1 查看文件内容

```
cat -n 显示行号
```

### 4.2 cp，mv组

| 命令 | 说明                                                         |
| ---- | ------------------------------------------------------------ |
| cp   | 复制，-r 要复制目录需要加上<br />-p 复制备份时保存文件原有属性<br />-a 是-rp的合体 |
| mv   | 1.移动，和cp用法一样<br />2.重命名 mv  /路径/原名 /路径/改后的名 |

注：他们俩都可以一次性移动多个文件或目录

### 4.3 echo与>、>>的作用

重定向符号系列

| 符号 | 说明                             |
| ---- | -------------------------------- |
| echo | 用法 echo 内容  >  文件路径      |
| >    | 重定向符号，先清空然后把内容写入 |
| >>   | 追加重定向符号，把内容写入到末尾 |

### 4.4 重定向符号固定搭配 cat 与 <<

向文件中写入多行内容

```sh
#示列
cat > /oldboy/oldboy.txt <<EOF
1
2
3
4
EOF

cat > 文件 << EOF
内容
EOF（表示结束）
```
