### 1.1 Linux下安装软件的方法

| Linux安装软件方法 | 说明                                                         | 对比                                         | 企业使用/选型                                                |
| ----------------- | ------------------------------------------------------------ | -------------------------------------------- | ------------------------------------------------------------ |
| yum/apt安装       | 自动下载软件包,自动解决依赖,完成安装,依赖网络(公网,局域网(需要特别搭建才能用)) | 点外卖,缺少的东西外卖解决.                   | 没有特殊要求的时候,首选yum/apt安装.<br />如果纯内网环境搭建内网yum/apt源. |
| rpm/dab包安装     | 本质与第1种类似,类似于windows.exe这样的文件.<br /<br/>yum/apt就是下载并替我们安装rpm包/deb包. | 速冻水饺.                                    | yum,apt无法使用，又没有内网yum源/apt源.<br临时使用rpm包或deb包. |
| 二进制安装        | 下载安装好的软件包,解压即用(需要简单配置),类似于绿色软件     | 自嗨锅,自热小火锅,自热米饭.依赖需要自己解决. | 软件没有rpm包,也无法yum/apt.<br /x<br/>不是所有软件都有二进制软件包. 部署数据库，网站服务tomcat |
| 源码安装          | 下载这个软件的源代码,通过三大步骤编译安装.<br/>配置(configure),编译(make),安装(make install) | 自己做饭.<br/>备菜,炒菜,上菜.                | 1.  /um/apt不行没有二进制的时候,编译安装.<br />2.  进行自定义安装(修改路径,修改里面的默认配置,增减默认的配置) |

### 1.2 使用光盘安装软件

```sh
# 挂载光盘
mount /dev/cdrom  /mnt/

# 检查是否挂载
df -h /mnt/

# 检查是否安装tree的软件包
rpm -qa | grep tree

# 如果有删除重新安装
rpm -e tree

# 使用光盘安装tree
rpm -ivh /mnt/Packages/tree-1.8.0-2.ky10.x86_64.rpm 

# 取消光盘
umount /mnt

# find的使用
find /mnt/ -type f -name "*.rpm" | grep tree
find 指定查找目录 指定类型 指定名字
-type f查找文件 d查找目录
```

### 1.3 yum-rpm搭配使用

#### 1.3.1 安装与检查

```sh
# 安装
yum -y install 软件名

# 检查
rpm -qa
-q  查询
-a  所有

# 检查指定软件包是否安装
rpm -qa | grep 软件包名
```

#### 1.3.2 查看软件包安装位置

```sh
# 检查是否安装
rpm -qa | grep 软件名

# 查看内容 
rpm -ql 软件名
```

#### 1.3.3 删除软件包

```sh
# 1.
rpm -e 软件包

# 2.
yum remove 软件包 ## 不要使用！！！！！！
```

#### 1.3.4 检查软件包变动的命令

```sh
rpm -aV 
# 结果说明
5 表示md5值变化（内容）
s 表示文件大小
M 表示
T 表示mtime修改时间
```
